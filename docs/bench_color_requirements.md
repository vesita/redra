# Bench 点云染色需求

## 当前用色

| 场景 | 材质名 | 当前 RGBA | 问题 |
|------|--------|-----------|------|
| 地面点 | `ground` | (0.42, 0.50, 0.35, 1.0) | 暗橄榄绿，低饱和不抢眼 |
| 墙面点 | `red` | (0.94, 0.35, 0.32, 1.0) | 可接受 |
| 待选/聚类点 | `yellow` / `cluster_n` | 各色 | 可接受 |
| 弃置点 | `blue` | (0.42, 0.38, 0.94, 1.0) | 略暗 |
| 非地面点 | `cyan` | (0.26, 0.80, 0.94, 1.0) | 可接受 |
| 背景原始点云 | `point_cloud` | (0.92, 0.90, 0.82, 1.0) | 暖白 OK |
| box | `disabled` | (0.4, 0.4, 0.4, 0.5) | 半透明灰 OK |

## 需求

添加以下新材质（均用于 3D 暗背景下，要求高亮度 + 高饱和度）：

### 语义色 — `materials/semantic/`

| 材质名 | 用途 | 建议 RGBA |
|--------|------|-----------|
| `ground` | **修改** 地面点云 | `[0.40, 0.95, 0.45, 1.0]` 亮绿色，区别于 `green` |
| `wall` | **新增** 墙面点云 | `[1.0, 0.35, 0.28, 1.0]` 亮红色（比 `red` 更亮） |
| `obstacle` | **新增** 障碍物 box | `[1.0, 0.85, 0.20, 1.0]` 亮黄色 |
| `cluster_bg` | **新增** 聚类背景 | `[0.70, 0.70, 0.70, 1.0]` 中性灰，不干扰聚类色 |

### 高亮色 — `materials/base/` 新增

| 材质名 | 用途 | 建议 RGBA |
|--------|------|-----------|
| `bright_blue` | 弃置/标记点 | `[0.20, 0.60, 1.0, 1.0]` 高亮蓝 |
| `bright_yellow` | 待选区 | `[1.0, 1.0, 0.20, 1.0]` 高亮黄 |
| `bright_cyan` | 非地面 | `[0.20, 1.0, 1.0, 1.0]` 高亮青 |

## 注册

在 `src/assets/materials.rs` `register_category` 中添加：

```rust
// semantic 分类新增 "wall", "obstacle", "cluster_bg"
// base 分类新增 "bright_blue", "bright_yellow", "bright_cyan"
```

各添加对应的 `.toml` 到 `assets/materials/semantic/` 和 `assets/materials/base/`。
